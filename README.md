# NHK vs 主要サブスク比較シミュレーター

NHK受信料を支払う代わりに、主要な動画配信サービス（サブスクリプション）をどれだけ利用できるかを可視化するコストパフォーマンス・シミュレーターです。

## 概要

このプロジェクトは、NHKの受信料（地上契約＋衛星契約の合計目安 ¥3,050/月）と同じ予算で、どのようなサブスク構成が組めるかをシミュレーションするために作成されました。

## 特徴

- **直感的な操作**: 利用したいサブスクをクリックするだけで、リアルタイムに合計金額とNHK受信料との差額を計算します。
- **モダンなデザイン**: Tailwind CSSとグラデーションを活用した、クリーンで視認性の高いインターフェース。
- **レスポンシブ対応**: PC、タブレット、スマートフォンなど、あらゆるデバイスで快適に利用可能です。

## 技術スタック

- **Frontend**: HTML5, Vanilla JavaScript
- **Styling**: Tailwind CSS (CDN版)
- **Visuals**: Lucide-like SVG Icons, Google Fonts (Noto Sans JP)

## 比較対象サービス (2026年時点目安)

- YouTube Premium
- Disney+
- hulu
- Apple TV+
- Netflix (Standard)
- Amazon Prime Video
- ABEMAプレミアム

## 使用方法

特別なサーバー設定やインストールは不要です。

1. このレポジトリをクローンまたはダウンロードします。
2. `index.html` を任意のブラウザで開きます。

## 免責事項

- 各サービスの料金は、シミュレーター作成時点の目安値です。実際の料金は各提供元の公式サイトをご確認ください。
- NHK受信料についても、支払い方法や契約形態により異なる場合があります。
